# Structured Data Extractor 🧩

A **Streamlit app** that converts raw, unstructured text into clean, structured JSON automatically. Perfect for receipts, invoices, or any text data.

---

## Features ✨

- Converts raw text into **structured JSON**  
- Automatically detects key fields like **names, dates, totals, items**  
- Handles **missing or uncertain values** gracefully  
- Ensures **consistent format** for all outputs  
- Easy-to-use **text input box** with clean interface
